解决vim内鼠标右键不能粘贴的问题
发表于2022-12-20|技术
debian9 开始，vim 右键会变成— (insert) VISUAL —的状态，没法粘贴很不方便。 解决办法： 1vim /usr/share/vim/vim80/defaults.vim 找到 set mouse 123if has('mouse') set mouse=aendif 将值从”a”改成”r” 123if has('mouse') set mouse=rendif 然后保存退出即可，下次启动 vim 即生效。
