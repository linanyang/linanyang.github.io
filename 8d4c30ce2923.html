vim 鼠标右键无法粘贴

平时使用 windows 系统，习惯使用鼠标右键进行粘贴，linux 系统里 vim 使用右键会默认变成(insert) 的状态，没法直接粘贴感觉很不方便。
解决办法：打开 vim 的配置文件，在 linux 中的路径一般为 /usr/share/vim/vimXX/defaults.vim 找到“set mouse=a”，将其修改为“set mouse=r” ，重启 vim 即可。
