解决vim内鼠标右键不能粘贴的问题

4月 1 2020 几秒 读完 (约 80 字)

RT debian9开始，vim右键会变成— (insert) VISUAL —的状态，没法粘贴很不方便。

解决办法：
vim /usr/share/vim/vim80/defaults.vim


查找 set mouse
if has(‘mouse’)
set mouse=a
endif


将值从”a”改成”r”
if has(‘mouse’)
set mouse=r
endif
然后保存退出即可，下次启动vim即生效
